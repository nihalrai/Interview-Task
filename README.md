# Careers Page Scraper

## Introduction

```shell

It crawls the careers page "https://www.cermati.com/karir" and print the result of job found on it in json

```

## Requirements

- Python 3.5+
- requests

## To run

- Unzip the file
- Traverse in folder
- Open Command prompt or Terminal
- Run "pip install -r requirements.txt"
- Run  "python main.py"
